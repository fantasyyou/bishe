matplotlib

tensorflow

pip install opencv-python -i https://mirrors.aliyun.com/pypi/simple/

pip install matplotlib -i https://mirrors.aliyun.com/pypi/simple/

pip install protobuf==3.6.0 -i https://mirrors.aliyun.com/pypi/simple/
